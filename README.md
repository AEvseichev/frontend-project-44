## Description
**Brain Games** is a collection of five command-line games based on popular logical puzzles. Each game asks questions that you need to answer correctly three times in a row to win.

---

## Installation

Before playing, make sure you have [Node.js](https://nodejs.org/) installed.

```bash
git clone [https://github.com/AEvseichev/frontend-project-44.git](https://github.com/AEvseichev/frontend-project-44.git)
cd frontend-project-44
make install
npm link
Games Logic & Demos
```

1. Brain Even

Determine if the number is even or not. Answer "yes" or "no".
[![asciicast](https://asciinema.org/a/qkVi7QpnzFgmycZ4.svg)](https://asciinema.org/a/qkVi7QpnzFgmycZ4)

2. Brain Calc

Solve a simple mathematical expression (addition, subtraction, or multiplication).
[![asciicast](https://asciinema.org/a/QeFoHxPibmUB67R8.svg)](https://asciinema.org/a/QeFoHxPibmUB67R8)

3. Brain GCD

Find the greatest common divisor of two given numbers.
[![asciicast](https://asciinema.org/a/4kwV5AmyQTtC4A4n.svg)](https://asciinema.org/a/4kwV5AmyQTtC4A4n)

4. Brain Progression

Find the missing number in an arithmetic progression.
[![asciicast](https://asciinema.org/a/4PkZtJvgpCAl9FLy.svg)](https://asciinema.org/a/4PkZtJvgpCAl9FLy)

5. Brain Prime

Determine if the given number is a prime number.
[![asciicast](https://asciinema.org/a/pw1jR5T3IpqYK1se.svg)](https://asciinema.org/a/pw1jR5T3IpqYK1se)

Usage
To start a game, use the following commands:

brain-even

brain-calc

brain-gcd

brain-progression

brain-prime
